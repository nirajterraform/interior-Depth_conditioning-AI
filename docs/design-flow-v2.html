API Endpoints

POST /api/detect-room-type

EXISTS — No changes needed

Auto-detects room type from uploaded image (bedroom, living room, kids room, kitchen, etc.)

POST /api/retrieve-catalogue

EXISTS — No changes needed

Retrieves products by room + theme. Already supports minPrice / maxPrice params for budget filtering.

POST /api/fal-inspire

EXISTS — No changes needed

Generates theme-styled room preview (~26s). Used in Step 2.

POST /api/fal-place-products

EXISTS — No changes needed

Places catalogue products in the room (~2–3 min). Accepts any product array — works for both Path A (all products) and Path B (user-selected subset).

POST /api/targeted-edit

EXISTS — No changes needed

Swaps individual products or applies text edits to generated room (~25–30s). Used in Step 4.

POST /api/cart (or Shopify API)

NEW — Needs implementation

Cart operations: add product(s), get cart total, checkout redirect. Platform TBD (Shopify / custom).
